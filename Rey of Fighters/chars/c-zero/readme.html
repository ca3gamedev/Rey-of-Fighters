北斗の拳より、羅将ハン…ではなく、クローンゼロ。
02UMでは「暗琉天破」だの「百人から先は覚えていない」だの、よりパロディ度が高まった。
彼の北斗アレンジとかが作られたら俺は歓迎する。
とは言っても俺は北斗はmugenでしか知らないが。wiki面白すぎ。

しかし、作ってて気付いたが、オリゼロはクローンに比べて色々と不遇だ。
ドットが少なかったり、白羅がどこでも判定でなかったり、声がおじいさん化したり、叛鏡（強バカめ）のドットで後ろの足先と手先が見切れてたり。
（ちなみに俺のオリゼロでは、クローンのを流用して直してある）

「ゼロを超えるものは、ゼロしかいない(ｷﾘｯ)」
なんなんだチョビヒゲ。クローンのくせに。

あと、叛鏡のドットをよーく見ると、スカートの翻し方が微妙に違っている。お暇な方はsffを見比べてみよう。
これの影響で、オリジナルとクローンでは「しゃがんでるみょんに当たるか否か」程度の赤枠の違いがある。
そんな違いつけるくらいならドット増やせよ。

オリジナルゼロと同様、意図的に弱体化させた部分が多くあるが、こちらもチートスイッチのボスモードをONにすることで原作相当に戻すことが出来る。
初期状態では、アーケードモードで敵として登場すると自動でボスモードになるようにしてある。

疾火煌陣（しっかこうじん）+K影を伸ばして相手を捕らえ、連続で打撃を加える。ガード不能。また、各打撃の直後にスパキャン可。
ちょっとアレッシーっぽくも見える。
魔舞紅躁（まぶこうそう）+Kおもむろに相手の目の前で屈み込み、立ち上がると、相手は既に葬られている、という技。屁ではない。
無敵投げ。少々発生まで間があり、間合いはかなり狭いが、つかみ判定が長時間持続するという、一風変わった性能。
疾火煌陣からのどこキャンで出すとまた別の効果になり、相手を硬直させ、好きなように追撃できる。
超必殺技：ゲージが1以上、またはMAX発動中に使用可
獄鎖斬陣+P出はかなり早く小パンからつながり、威力もなかなか高め。出だしには投げ以外への無敵もある。
ヒット数が多いため削りも凶悪（非ボスモードならヒットダメージと削りダメージが同じ）だが、ヒット、ガード問わず相手のゲージがモリモリ溜まってしまうという欠点も。
これにもガークラ効果あり。そして反確。
白羅滅精（はくらめっせい）+K（ほぼ）全画面技。普通は4ヒットだが至近距離だとヒット数が倍増し、ダメージも少し増える。
また、どこでも判定で、ダウン追い打ちも出来る。
ボスモードなら暗転から技が終わるまでずっと完全無敵というインチキ性能だが、非ボスモードだと無敵は飛び道具に対してのみになり、発生も原作より遅くなる。
MAX超必殺技：ゲージが3以上、またはMAX発動中でゲージが1以上のときに使用可
白羅滅精+KK8ヒット。こちらも近距離だとヒット数が倍。
MAX2：体力が3割以下、またはチートスイッチ「MAX2のHP制限解除」がONのであれば、MAX超必殺技と同じ条件で使用可
暗琉天破（あんりゅうてんは）+PP北斗琉拳と殺意の波動を組み合わせた、全く新しい奥義。魔闘気で空間を歪め、捕らえた相手に瞬獄殺っぽい攻撃を見舞うコマ投げ。
発生は1Fだが暗転後に掴む。間合いは常軌を逸脱していると言えるほどの広さ。非ボスモードではかなり狭くなるが、それでも相当広い部類。

チートスイッチについて

-2.stの中に
[state -2,チートスイッチ]
type=varset
trigger1=!var(0)
var(0)=～
という部分があり、ここを設定すると色々な強化を施すことができます。
基本的には俺tueeeeしてストレス解消するのにお使い下さい。

使い方は、var(0)=～のところを、使いたいチート機能の数を全て足した数に描き替えればOKです。足し算の式をそのまま書いても動作します。
PARとかやってる人とブール代数や2進数を知っている人は意味がわかると思います。

AIについて

このキャラにはAIが搭載されています。
レベルは1から8までの値を設定するように想定して作ってあります。レベル設定はc-zero.cmdの以下の部分で行えます。
また、;trigger28=1の行頭のセミコロンを外すと常時起動にできます。
[state -1,AIフラグセット]
type=varset
triggerall=!ishelper
trigger1=var(59)
trigger2=teamside=2||matchno>1
trigger2=ishometeam
～中略～
trigger27= numhelper(99999)
trigger27= helper(99999),var(59)=1
;trigger28=1;無条件起動
var(59)=1
;┃━┏┃AIレベル┃┓━┃…1から8まで
レベルの内容については、行動のランダム確率、切り返しの反応速度がレベルに比例するほか、3以下だと起き攻め、MAX発動が封印され、ぶっぱの確率が高くなります。
また、8だと一部のチートスイッチを理解した行動を行うようになるほか、
  • GM氏ギルのリザレクション
  • 無敵医師氏イノのメガロマニア
  • リン＆バット氏ホワイトのプーピーショット
に反応して専用対策モードに移行するようになります。
そもそもそこまでたどり着けないというのは秘密だッ！

中ポートレートについて

「ポトレ」フォルダの中には各種ライフバーに対応した中ポートレートが用意してあります。
余白や背景色の設定は最初から済んでいますので、座標(0,0)でファイル名通りの番号でsffに登録するだけでお使い頂けます。
ただし、9000-070.pngのみ、左と上に1ドットずつずらしてお使いください。

  • 9000-012(ﾑｴﾀｲE).png

  • 9000-078.png
はキャラ本体と同じパレットになっていますので、本体とパレット共有させることができます。

改変や転載などについて

デフォルトとは別のAIやパレット、アーケード用イントロ・エンディング等の作成と公開は大歓迎です。
sffとsnd以外のファイルからならデータ流用もOKです。
わざわざ許可を取ろうと連絡されても返事をするのが面倒なので、それはいりませんが、作ってから気が向いたら掲示板やスレなどで私に知らせてくだると嬉しいです。
なお、AIのついでに性能を調整するのも構いません。
特に性能を原作に近づける方向の改変の場合、本体に取り入れさせていただくようお願いする場合があると思います。
ただし、AIがONになると、またはAIレベルによって性能自体が変化する(AIにしかできない行動とか、無敵が増えるとか)というのは私の好みに反するのでできればやめて欲しいです。
また、対応する本体のバージョンも明記してくださるようお願いします。

改変や動画での使用などなども全てご自由にどうぞ。

転載は過去のバージョンも含め、ご遠慮ください。
ただし、何らかの理由で全バージョン公開が停止された場合は許可を取らなくても代理公開はOKとします。
