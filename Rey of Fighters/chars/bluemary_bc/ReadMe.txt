■はじめに■
	ダウンロードありがとうございます。
	このキャラはいろいろアレンジしてみたブルー・マリーです。
	餓狼っぽい要素も入ってたり。


■注意■
	このキャラの改変、転載はOKです。
	ただし自己責任でお願いします。
	感想とかバグ報告とかあると喜びます。


■設定など■
	"option.txt"というファイル内で、「ここから」「ここまで」と囲われた部分の
	設定を変えることが出来ます。


■簡単なシステム紹介■
	操作系は
		X		弱パンチ
		Y		強パンチ
		Z		ふっとばし
		A		弱キック
		B		強キック
		C		緊急回避、空中振り向き
		Start	挑発
		
		X+A		緊急回避、空中振り向き
		Y+B		ふっとばし
		Y+A		MAX発動、クイックMAX発動
		2+Y+A	ゲージタメ
	
	
	・ガードキャンセル
		ゲージが１本以上あると、ガード硬直中に地上ふっとばしをするとガーキャンとなる。
	
	・ジャンプ
		ジャンプには種類があり、歩きor走り、下上と入力、それぞれでの上入力の長さで
		小ジャンプ、通常ジャンプ、中ジャンプ、大ジャンプとがある。
	
	・MAX発動
		１ゲージを消費することで使用可能。
		効果時間中は攻撃力が上昇し、必殺技のキャンセル条件が変化。
	
	・クイックMAX発動
		通常技、特殊技動作中にMAX発動を行う。
		動作自体はなく、瞬時にニュートラルに戻るのでコンボ継続可能。
		ただし、２ゲージを消費する。
	
	・EX必殺技
		必殺技のコマンドのボタン部分を強弱同時押しにすることで、ゲージを消費するがより強力な必殺技が使用可能。
	
	・スーパーキャンセル
		必殺技をキャンセルしてゲージ技を使用可能。
	
	・ドリームキャンセル
		超必殺技を更に上位の超必殺技でキャンセルする。
		現状、M・スプラッシュローズから特定のもののみ対応。
	
	・カウンター
		相手の攻撃動作中に攻撃を当てるとカウンターとなる。
		相手の硬直が増え、受身不可になり、ダメージが増える。
