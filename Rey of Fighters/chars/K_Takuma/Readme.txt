; System
;================================================================================
Change between Normal and Another mode with the config.txt file

Throws: F or B + b/y (close range)
Roll: (can be done whilst guarding or performing a grounded normal, costs 1000 power): F or B a+b 
Knockdown Attack: (can be done whilst guarding, costs 1000 power): b+y or z
Super Cancel(SC): Perform a desperation move command during certain special moves (costs 1000 power).
MAX ON: a+y or c while neutral (costs 1000 power)
QUICK MAX: a+y or c during a normal move or command move (costs 2000 power)
