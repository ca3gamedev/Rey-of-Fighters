うっ！……ふう。

この改変パッチをあてると性能が変わります。AIも対応済みです。
当たり判定などは基本的に強化されていますが補正や一部の技は弱体化しています。


変更したのは主に以下の項目。

------------------------------------------------------------
：共通
------------------------------------------------------------

・大ポトレ＆小ポトレを荒れ狂う稲光のシェルミーに変更

・pal1～6を荒れ狂う稲光のシェルミー、pal7～12をノーマルシェルミーに変更

・各種イントロ対応キャラをすこし追加

・クイックMAX発動からのダッシュ入力を簡単に

・各種ジャンプ強攻撃ののけぞり時間をアフロンさんの社と同じに

・通常技などの下方向の当たり判定を強化
　バランスをとるために食らい判定も増加してます

・各種緊急回避の無敵時間を延長
　投げ無敵削除


------------------------------------------------------------
：荒れ狂う稲光のシェルミー
------------------------------------------------------------

・各種超必殺技の補正を高めに＆威力低下

・特殊技（６ｘ）が通常版とキャンセル版で性能変更
　通常版はふつうの中段技、キャンセル版は発生が速くなります。今まで通りコンボの繋ぎにどうぞ

・やたなぎのむち１段目ヒット時に相手が浮くように変更

・建御雷のゲージ増加率アップ

・無月の雷雲のダメージ低下


------------------------------------------------------------
：ノーマルシェルミー
------------------------------------------------------------

・コマンド投げに攻撃補正追加
　コンボからコマ投げを決めると威力が下がります

・強ダイヤモンドバストの移動距離アップ

・シェルミーカーニバルの無敵時間を延長

・Fカップチュード（当身投げ）を１Fに強化
　おっぱいをなぐるなんてとんでもない！



などなどです。
しょうもない改変すんな！という人はゴミ箱にポイしてください。


------------------------------------------------------------------------------
　　　　　　　　　　　　　　使い方と説明
------------------------------------------------------------------------------
普通に使う場合は、フォルダの中のduagi-AIフォルダをキャラ本体のフォルダに入れて、
一緒に入っている"Shermie(duagi).def"を"Shermie.def"に変更し上書きしてください。


他のAI製借者様と併用したい場合は、duagi-patchフォルダをキャラ本体のフォルダに入れて、
フォルダの中のShermie(duagi)をキャラ本体のフォルダに移してください。
data内に有るselect.defを開きShermie/Shermie(duagi).def
と登録すればおｋです。


;===========================================================================
AIの強さを１～10の間で変えることができます。

レベル6からはガードや技の反応が鋭くなってきます（デフォルトは10)。


変更の方法は、duagi-patchフォルダの中のConfig.txtファイルを右クリック→編集でひらいて
AIレベル＆各種設定用記述の、

[State -2,AI Level]
type = varset
trigger1 = var(59)=-1
;trigger2 = roundstate = 2 && ctrl

var(59) = 10;ここを設定
の　　　　↑
　　　ここの数字を変えて調節してください。

　　　対戦するときや動画を撮る時など、常時AIにしたいときは ";trigger2 = roundstate = 2 && ctrl" の左端の";"を消してください。



＊超反応うぜぇ！という人は設定で変更可能です。
　封印したい人は、Config.txtファイルのAIレベル＆各種設定用記述の下の記述、

;AI超反応スイッチ
[State -3,AI tc]
type = varset
triggerall = var(59)>0
trigger1 = 1

var(55) = 1;ここを設定

の　　　　↑
　　　ここの数字を変えて調節してください。1で超反応あり、0で超反応なしになります。
      デフォルトは超反応ありです。



＊AIの受け身をcanrecover時限定にするように設定ができます。
  設定したい人は、Config.txtファイルのAIレベル＆各種設定用記述の下の記述、

;AI受け身canrecoverスイッチ
[State -3,AI ukemi]
type = varset
triggerall = var(59)>0
trigger1= 1

var(40) = 1;ここを設定
の　　　　↑
　　　ここの数字を変えて調節してください。1でcanrecover無視、0でcanrecover時のみ受け身になります。
      デフォルトはcanrecover無視(本体記述そのまま)です。



＊勝利メッセージ表示のON/OFFが設定できます。
  設定したい人は、Config.txtファイルのAIレベル＆各種設定用記述の下の記述、


;勝利メッセージON/OFFスイッチ
[State -3,win m]
type = varset
trigger1= 1

var(41) = 1;ここを設定
の　　　　↑
　　　ここの数字を変えて調節してください。1で勝利メッセージあり、0で勝利メッセージなしになります。
      デフォルトは1(勝利メッセージあり)です。
